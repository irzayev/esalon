# eSalon

Веб-CRM для салона красоты: парикмахерские, косметология, маникюр, визаж и смежные услуги. Flask + SQLite + Tailwind, упакован в Docker.

## Возможности

- Управление заказами (записями) и процедурами
- CRM клиентов с историей визитов
- Расписание по **кабинетам** (рабочим местам)
- Онлайн-запись клиентов
- Услуги, пакеты, склад, зарплаты, финансы
- WhatsApp-уведомления и чат-бот
- Оплата Azericard, клиентский портал `/track`

## Быстрый старт

```bash
cp .env.example .env
# при необходимости отредактируйте .env

docker compose up -d --build
```

Приложение: http://localhost:7000

Учётка администратора из `.env` (по умолчанию `admin@esalon.local` / `admin123`).

## Структура домена

| Концепция | Описание |
|-----------|----------|
| **Клиент** | Имя, телефон, уровень лояльности, заметки |
| **Заказ** | Визит: процедуры, оплата, статус, расписание |
| **Кабинет** | Рабочее место (кресло, кабинет косметолога и т.п.) |
| **Услуга** | Процедура с ценой, длительностью и типом кабинета |
| **Сотрудник** | Мастер, назначаемый на заказ |

Типы кабинетов: парикмахерский, барбершоп, маникюр, косметология, визаж, массаж/SPA.

## Docker

- `esalon_web` — приложение (порт 7000)
- `esalon_data` — SQLite база
- `esalon_uploads` — загрузки (логотип, фото)

## Локальная разработка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export FLASK_ENV=development
export FLASK_APP=wsgi:app
flask run --port 7000
```
